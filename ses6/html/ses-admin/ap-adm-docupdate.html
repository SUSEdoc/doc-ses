C.1 Maintenance update of SUSE Enterprise Storage 6 documentation #Edit source

  • File Name: admin_docupdates.xml
  • ID: sec-adm-docupdates-6mu

  • Added a list of new features for Ceph 14.2.5 in the 'Ceph Maintenance Updates Based on Upstream 'Nautilus' Point Releases' appendix.

  • Added Section 26.8.5, “Archive Synchronization Module” (https://jira.suse.com/browse/SES-380).

  • Added Section 9.8, “RBD Mirroring” (https://jira.suse.com/browse/SES-235).

  • Added Book “Tuning Guide”, Chapter 8 “Improving Performance with LVM cache” (https://jira.suse.com/browse/SES-269).

  • Added /etc/ceph to the list of backup content in Section 3.1, “Back Up Ceph Configuration” (https://bugzilla.suse.com/show_bug.cgi?id=1153342).

  • Fixed targeting and re-weighting commands when adding new cluster nodes in Section 2.1, “Adding New Cluster Nodes” (https://bugzilla.suse.com/show_bug.cgi?id=1151861).

  • Added changelog entry for Ceph 14.2.4 (https://bugzilla.suse.com/show_bug.cgi?id=1151881).

  • Before running stage 5, verify which OSDs are going to be possibly removed in Chapter 2, Salt Cluster Administration (https://bugzilla.suse.com/show_bug.cgi?id=1150406).

  • Added two additional steps to the procedure that describes recovering an OSD in Section 2.9, “Recovering a Reinstalled OSD Node” (https://bugzilla.suse.com/show_bug.cgi?id=1137132).

  • Add a new chapter about Ceph Manager modules, focusing on the balancer module in Section 21.1, “Balancer” (https://bugzilla.suse.com/show_bug.cgi?id=1133550).

  • Rewrote the section about applying rolling updates of the cluster software and replaced stage 0 with manual commands to prevent an infinite loop in Section 2.12, “Updating the Cluster Nodes” (https://bugzilla.suse.com/show_bug.cgi?id=1134444).

  • Added a new section that describes how to replace the Admin Node: Section 2.10, “Moving the Admin Node to a New Server” (https://bugzilla.suse.com/show_bug.cgi?id=1145080).

  • Added a warning that enabling image journaling on all new images can have negative impact on cluster performance in Section 23.4.3, “Image Configuration” (https://bugzilla.suse.com/show_bug.cgi?id=1134734).

  • Added check for nscd process and a tip to keep clocks synchronized from the Active Directory controller (https://bugzilla.suse.com/show_bug.cgi?id=1144696).

  • Added Section 2.17, “Removing an Entire Ceph Cluster” and tips about purging custom roles in Section 6.6, “Adding Custom Roles” and Section 14.2, “User Roles and Permissions” (https://bugzilla.suse.com/show_bug.cgi?id=1138846).

  • Added a missing command in Section 20.1.1.3, “CRUSH Placement Rules” and added two new commands in Section 20.1.1.4, “Additional Commands” (https://bugzilla.suse.com/show_bug.cgi?id=1132696).

  • Added Section 2.7.2, “Removing All OSDs on a Host” (https://bugzilla.suse.com/show_bug.cgi?id=1142746).

  • Updated the output of ceph df command in Section 17.4, “Checking a Cluster's Usage Stats” (https://bugzilla.suse.com/show_bug.cgi?id=1143551).

  • Added two new advanced settings in Book “Deployment Guide”, Chapter 10 “Installation of iSCSI Gateway”, Section 10.4.5 “Advanced Settings” (https://bugzilla.suse.com/show_bug.cgi?id=1142341).

  • Added Section 2.16, “Deactivating Tuned Profiles” (https://bugzilla.suse.com/show_bug.cgi?id=1130430).

  • Added a statement that the cache tier migration method works only from a replicated pool in Section 22.3.2, “Migrate Using Cache Tier” (https://bugzilla.suse.com/show_bug.cgi?id=1102242).

  • Updated @/var/lib/ceph sub-volume creation in Section 33.6, “Btrfs Subvolume for /var/lib/ceph on Ceph Monitor Nodes” (https://bugzilla.suse.com/show_bug.cgi?id=1138603).
